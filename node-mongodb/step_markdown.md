# 全栈包安装
## Express 框架安装
1. 全局安装Express
`$ npm install -g express`
2. 最新express4.0 版本中将命令工具分家，工具命令如下
`$ npm install -g express-generator`
3. 查看版本
`$ express --version`
4. 当前项目链接全局安装包 npm link 包名
`$ npm link express`
